> 실습환경 : windows 10 home, git bash

### 버전관리 시스템(Version Control System)


- 소스코드의 중요한 변화들을 기록하는 행위

- 형상관리, 소스관리라고도 한다.


#### VCS의 효용

* Backup : 백업
* Recovery : 이전 상태로 복원
* Collaboration : 협업


대표적인 버전 관리 시스템으로는 CVS, SVN, GIT이 있다.


본질 : 변경사항을 관리한다

혁신 : CVS -> SVN -> GIT


Dropbox, Google Drive
- 일반인들이 사용하는 버전관리시스템


GIT은 개발에 반드시 필요하지만, 매우 어렵다.

현실의 코드는 훨씬 복잡하므로, git의 복잡성을 익혀 사용할만 하다.
